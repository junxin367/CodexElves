# 需求规格

## 原始需求摘要

去除概览顶部官方中转站推荐、推荐内容、Zed 相关功能、请作者喝咖啡功能。

## 领域术语

- 概览顶部官方中转站推荐：管理端概览页顶部推广卡片。
- 推荐内容：管理端推荐页、Codex++ 注入菜单里的推荐 tab、远程广告列表和对应桥接接口。
- Zed 相关功能：Zed Remote 打开、Zed 项目记录、Zed 设置项、Zed 专用桥接路由和测试。
- 请作者喝咖啡：赞赏 tab、二维码展示、内嵌赞赏图片和 README 赞赏区。

## 业务对象

- 管理端当前页面入口。
- Codex++ 注入增强菜单。
- 后端桥接路由。
- 当前 README 展示内容。

## 操作入口

- 管理端概览页不再展示官方中转站推荐。
- 管理端导航不再出现 Zed 远程项目、推荐内容。
- 注入菜单不再出现推荐内容、请作者喝咖啡、Zed Remote open。
- 后端桥接不再提供 `/ads` 和 `/zed-remote/*`。

## 功能行为

- REQ-001：概览页顶部官方中转站推荐不再展示。
- REQ-002：推荐内容不再展示、不再拉取远程广告列表、不再提供管理端或注入菜单入口。
- REQ-003：Zed 专用打开、项目记录、设置项、路由和命令不再提供。
- REQ-004：请作者喝咖啡入口、二维码资源注入和 README 赞赏展示不再提供。

## 状态规则

- 无新增业务状态。
- 已保存的旧设置字段不作为当前功能入口继续展示。

## 状态规则执行点

- 管理端 UI。
- 注入脚本菜单。
- 核心桥接路由。
- 文档展示。

## 权限规则

- 无新增权限规则。

## 异常场景

- 旧前端或外部调用继续请求 `/ads` 或 `/zed-remote/*` 时，桥接按未知路径处理。

## 非目标

- 不删除普通社区入口、用户脚本、会话删除、Markdown 导出、项目移动、Timeline、Upstream worktree、更新能力。
- 默认不清理历史 `CHANGELOG.md` 发布记录，见 CONFIRM-01。

## 回归保护

- Upstream worktree 的远程 SSH 项目解析能力必须保留。
- 当前通用设置保存、启动、用户脚本、模型目录、服务模式等非目标增强不应回归。

## 验收标准

- 仓库中当前 UI、注入脚本、桥接路由和 README 不再出现可用的推荐内容、赞赏或 Zed 功能入口。
- 相关 Zed / 推荐内容测试已删除或改为验证路由不可用。
- 格式检查和关键测试通过，或明确记录环境性失败。
